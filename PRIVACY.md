# Privacy Policy

**Backgammon** is designed to be private and offline.

### 1. Data Collection
This app does not collect, store, or transmit any personal information, location data, or usage statistics.

### 2. Third Parties
There are no third-party SDKs, advertisements, or analytics trackers included in the app.

### 3. Storage
Any game settings or preferences are stored locally on your device and are never uploaded to a server.

### 4. Contact
If you have any questions, please contact the developer via the GitHub repository.
